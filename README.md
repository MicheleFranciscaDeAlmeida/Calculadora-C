# Calculadora em C

## 📝 Descrição
Projeto de uma calculadora básica em C, implementando operações matemáticas fundamentais. 
Permite adicionar, subtrair, multiplicar e dividir números inteiros ou reais, demonstrando conceitos de **funções**, **condicionais (if/else)** e **loops**.

## ⚙️ Funcionalidades
- Soma, Subtração, Multiplicação e Divisão
- Tratamento de entradas inválidas
- Estrutura modularizada com funções

## 💻 Tecnologias
- Linguagem: C
- Compilador: GCC
- Conceitos aplicados: funções, modularização, condicionais, loops

## 🚀 Como executar
1. Instale o GCC:  
   - Windows: [MinGW](https://www.mingw-w64.org/)  
   - Linux: `sudo apt install gcc`  
   - MacOS: `brew install gcc`  

2. Clone o repositório:  
```bash
git clone https://github.com/SEU-USUARIO/calculadora-c.git
cd calculadora-c/src
Compile e execute:
gcc main.c operacoes.c -o calculadora
./calculadora

3. Compile e execute:
gcc main.c operacoes.c -o calculadora
./calculadora
